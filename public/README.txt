【写真の入れ方】
このフォルダ（public）に、下のファイル名で写真を置くと、サイトの各所に表示されます。
※ファイル名は半角英語で、拡張子は .jpg にしてください（.jpeg でも可）。

  hero.jpg   … トップの背景（夜の店内）※設定済み
  about.jpg  … 「序」の内観写真 ※設定済み
  night.jpg  … 「夜」の写真 ※設定済み
  story.jpg  … 「由縁」の写真（未設定・入れると表示）
  day.jpg    … 「昼」の写真（未設定・入れると表示）
  dango.jpg  … メニューの団子写真（未設定・入れると表示）

未設定のものは、写真を置くだけで自動表示されます（コード編集不要）。
差し替えたいときは、同じ名前で上書きアップロードすればOKです。
